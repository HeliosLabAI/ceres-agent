Setup Guide

Choose your preferred AI model setup

RECOMMENDED

🏠 Local Ollama Setup

Run AI models locally on your machine. Fast, private, and works offline.

  • 1
    Install Ollama
    Download from ollama.com and run the installer
  • 2
    Download a Model
    ollama run llama3.2
  • 3
    Start Ceres AI Agent
    Open the app - it will automatically detect Ollama models

☁️ Cloud API Setup

Use cloud-based AI models. No local GPU required, works on any machine.

  • 1
    Get API Key
    Sign up at OpenAI, Anthropic, or other providers
  • 2
    Configure API
    Set your API key in Settings → API Configuration
  • 3
    Start Chatting
    Select your cloud model and start using Ceres AI
